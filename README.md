# ArduinoClock
Hobby Project: Table clock using Arduino (No RTC Module)

Components:

> Arduino Mega ATMEGA2560
> 16x2 LCD Display (JHD204A)

Connections:

  LCD             Arduino
  1                 GND
  2                 5V
  3               Pin 2
  4               Pin 3 
  5               Pin 4
  6               Pin 5
  11              Pin 11
  12                GND
  13              Pin 12
  15                5V  
  16                GND
